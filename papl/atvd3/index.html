EXPLICANDO TAGS

por: Paula Malisia Moreira

HTML

HTML é uma etiqueta que nomeia os elementos de uma página, de onde ele começa até onde ele termina.

HEAD

Serve para indicar ao navegador onde está o cabeçalho da página.

BODY

É tudo o que aparece no site, todo o corpo dele.

HEADER

Representa o cabeçalho do site.

SECTION

A section determina o agrupamento em uma sessão lógica da página.

DIV

Define uma divisão ou sessão e um documento, podendo ser até mesmo dentro de uma sessão.

FOOTER

É a marcação do rodapé da página.

H1 até H7

Os Hs servem para determinar a importância do texto dentro do site, sendo que 1 é o mais e 7 o menos.
Lembrando que apesar dessa formação diminuir e aumentar o texto dentro da página, ela apenas determina a importância, pois o que mexe no tamanho é o css.

HR

Representa uma linha de um texto, sessão, div, etc.

P

Marca um novo parágrafo.

P

É o parágrafo, normalmente depois do H.

UL

Faz uma lista não ordenada.

  • Primeiro Elemento da lista
  • Segundo elemento da lista...

LI

Representa cada tópico da lista.

BR

Pula uma linha.

SPAN

Funciona como uma div, porém para texto.

B

Transforma o texto em negrito.

I

Transforma o texto em itálico.

A

Tag de link de HTML.

IMG

Adiciona uma imagem e junto a essa tag, utilizamos o atributo scr="" que deve ser digitado onde a imagem se encontra.

VIDEO

Adiciona um video e junto a essa tag, utilizamos os atributos source scr="" que deve ser digitado onde o video se encontra e type o tipo do video.

AUDIO

Funciona da mesma forma que a tag video.
Adiciona um audio e junto a essa tag, utilizamos os atributos source scr="" que deve ser digitado onde o audio se encontra e type o tipo do audio.

MAIN

Define como o conteúdo principal dentro do body.

NAV

Representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.

CODE

Apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código.
